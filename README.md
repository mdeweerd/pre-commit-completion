# Bash completion for pre-commit

## Usage

1. Get pre-commit-completion.bash
2. Source it into your shell (`.bashrc` or `.bash_profile`):

   ```bash
   source path/to/pre-commit-completion.bash
   # or
   . path/to/pre-commit-completion.bash`
   ```
